# XJAVPlayer
  本播放器是根据AVPlayer进行封装的，具有以下功能：
  </br>

1.一句代码就能调用播放 
</br>
2.支持开始/暂停  
</br>
3.支持放大/缩小  
</br>
4.支持随屏幕旋转 
</br>
5.支持拖拽进度 
</br>
6.时间显示 
</br>
