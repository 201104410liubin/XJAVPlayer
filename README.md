# XJAVPlayer
  本播放器是根据AVPlayer进行封装的，具有以下功能：

1.一句代码就能调用播放 
2.支持开始/暂停  
3.支持放大/缩小  
4.支持随屏幕旋转 
5.支持拖拽进度 
6.时间显示
